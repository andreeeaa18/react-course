# Curs React

Acesta este un curs complet de React construit cu Slidev.

## Cum să începi

Pentru a porni prezentarea:

```bash
pnpm install
pnpm dev:hub
```

Vizitează http://localhost:3030

### Module disponibile

Cursul este organizat în module independente. Pentru a rula un modul specific:

- **Hub (navigare)**: `pnpm dev:hub` - http://localhost:3030
- **Introducere**: `pnpm dev:introduction` - http://localhost:3031

Editează fișierele `slides.md` din fiecare modul pentru a vedea modificările.
