# About This Course

This is a **live, interactive** React course that runs entirely in your browser.

- 🔥 **Live Code Editors** - Write and run React code instantly
- 📦 **Sandboxed Execution** - Safe, isolated runtime environment
- 🎯 **Module-Based** - Independent, lazily-loaded presentations
- 🚀 **Modern Stack** - Built with Slidev, Vite, and Sandpack
